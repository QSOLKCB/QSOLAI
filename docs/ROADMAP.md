# Roadmap

## v0.1.x — hardening only

- expand malformed-input and cross-platform ZIP known-answer vectors;
- add schema publication artifacts derived from the frozen contracts;
- exercise ManualAdapter multi-slot import workflows more deeply; and
- improve bounded subprocess streaming without expanding its authority.

## Deferred profiles

The names `READ_ONLY_EXTERNAL`, `WORKSPACE_WRITE` and `CONTROLLED_EXECUTION` are reserved. They require separate threat models, capability semantics, audit rules and explicit owner approval before implementation.

## Explicitly out of scope

- chatbot or web UI;
- provider SDK integration in the core;
- autonomous action execution;
- probabilistic final adjudication;
- hidden retry loops or mutation state;
- deterministic-model-inference claims;
- consciousness or AGI claims; and
- replacing human medical, legal, financial or mission-critical authority.
